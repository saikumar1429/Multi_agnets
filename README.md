# Multi_agnets
🤖 Multi-Agent AI System

A Multi-Agent AI system built with Streamlit that analyzes uploaded documents using multiple AI agents working collaboratively.

---

## 🚀 Features

- 📄 Upload a text document
- 🤖 Multiple AI agents analyze the document
- 🧠 Each agent provides a unique perspective
- ⚡ Clean and interactive Streamlit UI
- 🧩 Coordinator pattern to manage agent workflow

---

## 🏗️ Architecture

User → Streamlit UI → Coordinator → Multiple Agents → Aggregated Results

The `Coordinator` class orchestrates multiple AI agents and collects their outputs before displaying them in the UI.

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Multi-Agent Architecture
- Prompt Engineering
